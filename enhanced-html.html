User Guide

Installation Guide

  1. Site Preparation

    Prepare a level, waterproof area near the wastewater source, with access to electricity.

  2. Plumbing Connection

    Connect inlet pipe to the wastewater source and outlet pipe to collection tank.

  3. Electrical Setup

    Connect the control unit to a stable 220V power source with proper grounding.

  4. Sensor Calibration

    Follow the calibration steps in the manual for pH and salinity sensors.

Operation Guide

  1. System Startup

    Press the power button and wait for the green indicator light before proceeding.

  2. Initial Priming

    Run the system with clean water for 5 minutes to prime all components.

  3. Processing Mode

    Select "Auto" mode for standard operation or "Manual" for custom settings.

  4. Monitoring

    Check the digital display for real-time readings of flow rate, pH, and salinity.

Maintenance Guide

  • Daily Check

    Inspect for leaks and check sensor readings on the digital display.

  • Weekly Cleaning

    Perform backwash cycle by pressing the "Backwash" button on the control panel.

  • Monthly Maintenance

    Clean external filters and inspect electrode conditions in the ionization chamber.

  • Quarterly Maintenance

    Full system inspection including all valves, seals, and sensor calibration.

Component Replacement Schedule

Ionization Electrodes

Replace: Every 6-8 months

Signs: Decreased efficiency, visible corrosion

Coagulation Media

Replace: Every 3-4 months

Signs: Cloudy filtered water, slow flow rate

UV Lamp

Replace: Every 12 months

Signs: Indicator light on control panel

Pre-filter Element

Replace: Every 2-3 months

Signs: Reduced flow rate, visible debris

pH Sensor

Replace: Every 12-18 months

Signs: Inconsistent readings, calibration failure

Pump Seals

Replace: Every 12 months

Signs: Noise during operation, leakage
